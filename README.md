# 📸 CodeSnap Pro

GitHub depolarınızdaki veya doğrudan yapıştırdığınız kodları, harika ve paylaşılabilir PNG resimlerine dönüştüren modern bir Streamlit web uygulamasıdır.

## 🚀 Özellikler

* **🐙 GitHub Entegrasyonu:** Herhangi bir genel (public) GitHub repo linkini girerek dosyalarınızı anında listeleyin ve yükleyin.
* **🎨 Tema ve Dil Desteği:** Pygments altyapısı ile kodlarınızı Monokai, Dracula gibi popüler temalarla renklendirin.
* **🌈 Gelişmiş Görsel Stiller:** İster sade, ister temalı, isterseniz de şık degrade (gradient) arka planlar ekleyin.
* **☁️ Bulut Uyumu:** Tamamen saf Python (Pillow & Pygments) kullanılarak optimize edilmiştir, harici hiçbir bağımlılık gerektirmez.

## 🛠️ Kullanılan Teknolojiler

* **Streamlit:** Web arayüzü ve kolay dağıtım için.
* **Pillow (PIL):** Güvenli ve hızlı görüntü işleme için.
* **Pygments:** Kod sözdizimi renklendirmesi (syntax highlighting) için.
* **Requests:** GitHub API entegrasyonu için.
